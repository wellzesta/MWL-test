# MyWellzestaLife

# A sample app using ember-cli

This is a sample application created using ember-cli.

## Installation

* `git clone https://github.com/wellzesta/MWL-test.git`
* `cd MWL-test`
* `npm install && bower install`

## Running

* `ember server`
* Visit `http://localhost:4200`

## Running Tests

* In browser
	* `ember server`
	* Visit `http://localhost:4200/tests`
* Command line 
	* `ember test`, OR
	* `ember test --server`

## Building

* `ember build`
